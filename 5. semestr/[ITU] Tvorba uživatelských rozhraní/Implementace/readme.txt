Pro zprovoznění je potřeba:
	1) rozbalit ZIP archiv
	2) nahrát soubory společně s požadovanými knihovnami na web server
	3) spustit primární soubor index.php, po jeho spuštění je dostupná webová aplikace


Popis projektu:
	Účelem našeho projektu byla jednoduchá a intuitivní webová aplikace pro desktopové systémy, která slouží pro evidenci výpočetní techniky.
	Naším zaměřením byly základní školy. 
	V aplikaci je možné
		- přihlásit se/odhlásit se
		- spravovat výpočetní techniku
		- spravovat opravy techniky
		- spravovat zaměstnance školy
		- zobrazovat zařízení v prostorách školy
		- zobrazovat statistiky
	Aplikace je implementována v HTML & CSS s využitím Javascriptu, PHP a výše zmíněných knihoven.
